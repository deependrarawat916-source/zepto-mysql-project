# Zepto Data Analysis using MySQL

## Overview
This project is an **entry-level MySQL data analysis project** based on a fictional **Zepto grocery delivery system**.  
It demonstrates SQL fundamentals required for **data analyst and backend internships**.

---

## Objectives
- Create a relational database
- Analyze business data using SQL
- Practice real-world queries
- Build a portfolio-ready SQL project

---

## Database Tables
- customers
- products
- orders
- order_items
- payments

---

## Tools & Technologies
- MySQL
- SQL
- MySQL Workbench / phpMyAdmin

---

## Key Queries
- Total revenue calculation  
- Customer-wise orders  
- Top-selling products  
- Daily revenue analysis  
- Repeat customer identification
